PROJECT TITLE: Treasure game
PURPOSE OF PROJECT:Randomized spots that either have a reward or a troll.
The troll button will randomly take off points and the reward will randomly give you points
It will show you how many rewards you have left and how many found.
You will have only 50 tries to play the game.
It will update the last thing that happened to you, either trolled or rewarded.
VERSION or DATE: 12/15/2019
HOW TO START THIS PROJECT:
